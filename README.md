# EscapeFromLab

## Особенности

- Управление параметрами: здоровье, тревога, доверие к ORION
- Инвентарь и предметы
- Система флагов и посещённых событий
- Условия доступности выборов
- Несколько концовок
- Лимит ходов

## Архитектура

- `GameState` — центральное состояние игры
- `Scene` и `Choice` — сцены и варианты действий
- `Effect` — интерфейс эффектов
- `Condition` — интерфейс условий
- `Script` — сборка всех сцен и переходов
- Вспомогательные классы `Ef` и `Req` для удобного создания эффектов и условий

## Тестирование

Проект включает модульные тесты (unit-тесты) для проверки логики состояния, эффектов и условий.
